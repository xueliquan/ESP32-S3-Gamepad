# 硬件设计文档

## 元器件清单

### 核心组件
1. ESP32-S3-WROOM-1-N8R2 模块
2. 3.7V 锂电池 (2000mAh+)
3. USB Type-C 充电模块 (TP4056)
4. 稳压模块 (AMS1117-3.3V)

### 输入设备
1. 按钮 x17 (6x6x5mm 贴片微动开关)
2. 双轴摇杆 x2 (ALPS RKJXV1224005)
3. 模拟扳机 x2 (可选)

### 输出设备
1. 震动电机 x2 (1020型扁平电机)
2. LED指示灯 x4 (0805封装)
3. 电池电量指示灯 x1 (三色LED)

### 连接器
1. USB Type-C 插座
2. 电池连接器 (JST PH2.0)
3. 编程接口 (6pin SWD)

## PCB设计要点

### 电源部分
```

锂电池 (3.7V) → TP4056 → AMS1117 → 3.3V系统
↓
USB充电

```

### 布局建议
1. ESP32模块居中放置
2. 摇杆对称布局
3. 按钮按人体工学排列
4. 震动电机在握把位置
5. 远离天线区域

### 走线规则
- 电源线宽度: 20mil (1A), 40mil (2A)
- 信号线宽度: 6mil
- 最小间距: 6mil
- 天线周围净空区: 15mm

## 故障排除

### 常见问题
1. 无法充电：检查TP4056连接
2. 按钮不响应：检查上拉电阻
3. 摇杆漂移：重新校准
4. 蓝牙不稳定：检查天线位置
5. 耗电过快：检查睡眠模式
